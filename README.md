# Interactive-Image-Gallery

📸 Interactive Image Gallery
A modern and interactive image gallery built using HTML and CSS, where images are displayed in grayscale by default and dynamically transition to full color on user interaction (hover/click).

This project focuses on enhancing user experience through smooth visual effects and clean UI design.

✨ Key Features
🎨 Grayscale to color transition effect ⚡ Smooth and responsive hover interactions 🖼️ Clean and minimal gallery layout 💻 Lightweight and fast performance 📱 Responsive design (works across devices)

🛠️ Tech Stack
HTML5 – Structure of the gallery CSS3 – Styling, filters, and transitions

⚙️ Implementation Details
The project uses the CSS filter property to apply a grayscale effect to images. On hover, the filter is removed using smooth CSS transitions, creating an engaging visual interaction.

📂 Project Structure
Interactive-Image-Gallery/ │── index.html │── style.css │── images/

🚀 Getting Started
Clone the repository Navigate to the project folder Open index.html in any web browser
